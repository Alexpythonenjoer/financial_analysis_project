# Финансовый анализ транзакций и поиск по документам

Проект выполнен в рамках тестового задания и состоит из двух частей:
1. **Анализ финансовых транзакций** – очистка, исследование, визуализация и прогнозирование спроса.
2. **Разработка поисковика** – краулинг файлов различных форматов (docx, xlsx, pdf, txt, архивы), извлечение текста и загрузка в базу данных с полнотекстовым поиском.

## Установка и запуск

### 1. Клонирование репозитория и подготовка окружения

```bash
git clone <https://github.com/Alexpythonenjoer/financial_analysis_project>
cd financial_analysis_project
python -m venv .venv
source .venv/bin/activate      # Linux/Mac
# .venv\Scripts\activate       # Windows
pip install -r requirements.txt
```
### 2. Запуск анализа транзакций (Задание 1)
````
bash
python src/run_analysis.py
Скрипт выполнит:

Очистку данных (удаление отрицательных сумм, некорректных дат, пропусков).

Расчёт топ‑5 услуг по количеству заказов.

Расчёт средней суммы транзакций по городам.

Определение услуги с максимальной выручкой.

Долю способов оплаты.

Выручку за последний месяц.

Объединение с данными клиентов и категоризацию по уровню капитала.

Построение графиков (распределение сумм, выручка по услугам, зависимость от возраста).

Прогнозирование спроса на следующий месяц с помощью линейной регрессии.

Результаты сохраняются в папку output/:

Графики – output/charts/

Прогноз – output/forecast.csv

Также все выводы дублируются в консоль.
````

### 3. Запуск поисковика по документам (Задание 2)
4.1. Генерация тестовых файлов
````
bash
python crawler/generate_docs.py
Будут созданы файлы разных форматов (docx, xlsx, pdf, txt, zip) в папке crawler/storage/.
````
4.2. Краулинг и извлечение текста
````
bash
python crawler/crawler.py
Скрипт обходит все файлы в crawler/storage/ (включая вложенные папки и архивы), извлекает текст из поддерживаемых форматов и сохраняет результат в crawler/output/file_index.csv.
````
4.3. Загрузка в базу данных и поиск
````
bash
python crawler/db_loader.py
Создаётся SQLite база данных crawler/output/database.sqlite с полнотекстовым индексом (FTS5). После импорта вы можете ввести слово для поиска – программа покажет файлы, содержащие это слово, с фрагментами текста.
````


### Результаты анализа (выборочные)
````
Топ‑5 услуг по заказам: Инвестиционное консультирование (2085 заказов), Управление активами (1729), Налоговое планирование (1316), Финансовое планирование (1315), Структурирование капитала (1287).

Средний чек варьируется от 53 000 до 58 000 по разным городам.

Максимальную выручку приносит «Инвестиционное консультирование» – более 104 млн.

Способы оплаты: кредитная карта (~40%), банковский перевод (~35%), неизвестно (~10%), наличные (~10%), криптовалюта (~5%).

Выручка за последний месяц составила ~103,5 млн.

Клиенты с высоким капиталом (>1 млн) генерируют 319 млн выручки, что значительно больше, чем другие категории.

Прогноз на следующий месяц – около 3185 транзакций.

Примечания
Для работы с RAR-архивами требуется установленная утилита unrar. Если её нет, скрипт автоматически создаёт ZIP-архивы вместо RAR.

Все пути в скриптах относительные – запуск должен производиться из корневой папки проекта.

При первом запуске run_analysis.py автоматически создаются необходимые подпапки в output/.
````
